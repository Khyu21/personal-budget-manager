# Personal Budget Manager

## Overview

A Python application that helps users analyze their monthly finances by calculating disposable income, utility costs, purchasing power, and transaction validity.

## Features

* Monthly budget analysis
* Disposable income calculation
* Water bill estimation
* Purchasing power assessment
* Transaction validation
* Budget history tracking

## Technologies Used

* Python 3

## How to Run

1. Download the project files.
2. Open the project in VS Code or another Python IDE.
3. Run `personal_budget_manager.py`.
4. Follow the prompts to generate budget reports.
