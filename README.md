## Overview
This Python application was developed as a practical assignment to simulate a personal finance tracker. The program assists users in analyzing monthly finances by automating the calculation of disposable income, estimating utility costs (specifically water bills), assessing purchasing power, and validating transaction entry.

## Features
* **Financial Calculations:** Automatically computes disposable income and total purchasing power based on user inputs.
* **Utility Cost Estimation:** Features built-in logic to approximate monthly water bill expenses.
* **Data Validation:** Ensures all transaction inputs are valid and within safe budget thresholds to prevent accidental overspending.
* **History Tracking:** Logs budget entries sequentially to monitor financial trends over time.

## Technologies Used
* **Language:** Python
* **IDE:** Developed and tested using PyCharm.

## How to Run
1. Download or clone the repository containing `personal_budget_manager.py`.
2. Open the project folder inside **PyCharm**.
3. Right-click anywhere inside the `personal_budget_manager.py` file script editor and select **'Run'**, or click the green play button in the top right corner.
4. Follow the interactive terminal prompts at the bottom of PyCharm to input financial data and review the generated budget summary.

## Future Enhancements
If extended beyond the scope of this practical, the application could be improved with the following features:
* **Expense Categorization:** Implementing systems to sort transactions into specific categories (e.g., food, utilities, entertainment).
* **Savings Goal Tracker:** Adding a module to help users set and track progress toward specific financial targets.
* **Data Persistence:** Integrating CSV export capabilities to save budget history externally for analysis in Excel.
* **Graphical User Interface (GUI):** Transitioning the project from a command-line interface to a visual dashboard.
